네이버 블로그에 정리한거 다시 정리하기!!

###Apache Tomcat이란?
세계에서 가장 많이 사용되는 WAS(Web Application Server)이다.
컴퓨터에 운영체제를 설치해야만 컴퓨터를 사용할 수 있는 것처럼, 자바를 이용하여 작성된 웹 어플리케이션은 WAS가 있어야만 실행 가능하다.


간단한 웹 어플리케이션 만들어보기

new -> servlet -> 이름 Helloservlet으로 설정해줌
http://localhost:8080/{프로젝트이름}/{URL Mapping값}
http://localhost:8080/firstweb/Helloservlet

위의 코드처럼 간단한 예제를 수행해보았다
doGet()메서드만 수정해주었다

실행결과


doGet() 메서드

웹서버가 get메서드 방식으로 요청을 보낼때 서블릿에 doGet()메서드가 호출이 된다
get 메서드는 웹브라우저가 서버에게 문서를 요청할때 사용하는 방식


크롬 브라우저에서 보는법

Window -> Web Browser -> Chrome
으로 설정하면 크롬에서 확인이 가능하다
