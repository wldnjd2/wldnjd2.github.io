List와 Numpy의 차이점

연산에서의 차이점이 있다
- List
A = [1,2,3]
B = [4,5,6]
A + B 일때 결과는 [1,2,3,4,5,6]

- Numpy
import numpy as np
A = [1,2,3]
B = [4,5,6]
np_A = np.array(A)
np_B = np.array(B)
np_A + np_B 의 결과는 array([5,7,9])

Numpy 기본 함수들

1. arange 메소드
파라미터로 받은 리스트를 반환해주는 메소드
( )괄호 안의 값이 1개일때와 여러개일때의 의미가 조금씩 다르다.

2. zeroes, ones 메소드

  • zeros() 메소드
    0으로 초기화된 배열 객체를 반환하는 메소드

    함수는 1로 초기화된 배열 객체를 반환하는 메소드
